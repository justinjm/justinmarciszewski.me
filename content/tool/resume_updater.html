How do I get an API key?
  1. Go to Google AI Studio → API Keys
  2. Sign in with your Google account
  3. Click Create API Key and select a project (or create one)
  4. Copy the key and paste it above

The free tier includes generous rate limits for Gemini Flash models. Your key stays in your browser and is never sent anywhere except Google's API.
